## 🐛 常见问题

### Q: 输入"生成图片"后没有反应？
**A:** 检查以下几点：
1. 确保消息包含关键词："生成图片"、"画一张"或"生图"
2. 检查浏览器控制台是否有错误
3. 确认开发服务器正在运行

### Q: 图片不显示？
**A:** 
1. 检查网络连接
2. 图片URL可能需要时间加载
3. 打开浏览器开发者工具查看Network标签

### Q: 快捷按钮点击后没有填充文本？
**A:** 
- 快捷按钮会自动填充提示词到输入框
- 需要手动点击发送按钮或按Enter

## 📚 历史对话功能

### 功能说明
Agent现在支持多会话管理和历史记录功能，所有对话会自动保存到浏览器本地存储。

### 测试步骤

1. **查看历史对话列表**
   - 点击对话框左上角的">"按钮
   - 侧边栏会滑出显示历史会话列表

2. **创建新对话**
   - 在历史对话侧边栏点击"新建对话"按钮
   - 系统会创建一个新的空白对话
   - 新对话标题默认为"新对话"

3. **自动标题生成**
   - 在新对话中发送第一条消息
   - 对话标题会自动更新为消息的前20个字符

4. **切换对话**
   - 在历史列表中点击任意对话
   - 主聊天区域会切换到该对话
   - 侧边栏自动收起

5. **删除对话**
   - 鼠标悬停在历史列表的对话项上
   - 右上角会显示删除按钮（垃圾桶图标）
   - 点击删除按钮，确认后删除
   - 如果删除的是当前对话，会自动切换到其他对话

6. **数据持久化**
   - 刷新页面后，历史对话仍然存在
   - 关闭浏览器重新打开，数据依然保留
   - 数据存储在localStorage中

**存储键名：**
- `agent_chat_sessions` - 所有会话数据
- `agent_current_session` - 当前会话ID

**数据结构：**
```typescript
interface ChatSession {
  id: string;           // 会话ID
  title: string;        // 会话标题
  messages: Message[];  // 消息列表
  createdAt: number;    // 创建时间
  updatedAt: number;    // 更新时间
}
```
