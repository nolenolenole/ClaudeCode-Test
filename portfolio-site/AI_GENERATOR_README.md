# AI Generator UI 组件实现文档

## 概述

基于设计稿 `d2c_8e459a` 实现的AI图片生成工具界面，包含完整的UI组件和交互功能。

## 技术栈

- **框架**: React + TypeScript
- **样式**: Tailwind CSS
- **图标**: lucide-react
- **动画**: framer-motion
- **路由**: react-router-dom

## 组件结构

### 1. 侧边导航栏 (Sidebar)
- **位置**: 左侧固定，宽度88px
- **背景**: 渐变色 `from-[#F9FFFE] via-[#F2F3FE] to-[#FDF9FE]`
- **功能**:
  - Logo展示（带BETA标识）
  - 菜单导航（发现、生成、资产、画布）
  - 积分入口（带渐变边框）
  - 用户头像（渐变背景）
  - 消息和设置按钮

**关键特性**:
- 菜单项支持选中状态切换
- 选中项显示白色背景和紫色图标
- 积分入口使用CSS渐变边框技术

### 2. Tab切换组件 (TabSwitch)
- **功能**: 图片生成/视频生成模式切换
- **样式**: 圆角白色背景，选中项高亮
- **交互**: 点击切换，带过渡动画

### 3. 生成区域
包含以下子组件：

#### 图片上传
- 虚线边框的上传区域
- 悬停时边框颜色变化
- 支持点击上传

#### 提示词输入
- 多行文本输入框
- 占位符文本："请输入你的创意..."
- 自适应高度

#### 参数选择器
- **模型选择**: Qwen Image Turbo
- **图片参数**: 智能比例 | 高清2K | 1张
- **联想助手**: 辅助创意生成

#### 操作按钮
- **翻译按钮**: 紫色文字，白色背景
- **生成按钮**: 
  - 未输入时灰色禁用状态
  - 输入后紫色可用状态
  - 显示所需积分数

### 4. 运营位
三个卡片式布局：
- **无限画布上新**: 深色背景，480px宽
- **AI视频创作计划**: 浅紫色背景，300px宽
- **Wan2.6模型上新**: 浅紫色背景，300px宽

### 5. 灵感广场 (Gallery)
- **布局**: 6列网格布局
- **功能**: 
  - 分类筛选（全部、图片、视频、无限画布）
  - 图片卡片展示
  - 悬停效果（渐变遮罩 + 操作按钮）

#### 图片卡片 (ImageCard)
- **悬停效果**: 
  - 缩放动画 (scale: 1.02)
  - 底部渐变遮罩显示
  - 作者信息和操作按钮
- **操作按钮**: 
  - 做同款（Copy图标）
  - 图片编辑（Edit图标）

## 设计规范

### 颜色系统
```css
--primary: #6928FE
--background-gradient: linear-gradient(180deg, #F9FFFE, #F2F3FE 37%, #FDF9FE)
--card-gradient: linear-gradient(130deg, #BBA6FF, #AADCFE 50%, #DFC2FF 89%)
--avatar-gradient: linear-gradient(143deg, #D6F482 10%, #AAE1FE 26%, #AF83FD 49%, #5BF2C6 86%)
```

### 间距规范
- 侧边栏宽度: 88px (w-22)
- 小间距: 4px (gap-1)
- 中间距: 8px (gap-2)
- 大间距: 12px (gap-3)

### 圆角规范
- 小圆角: 4px (rounded)
- 中圆角: 8px (rounded-lg)
- 大圆角: 12px (rounded-xl)
- 超大圆角: 24px (rounded-3xl)
- 完全圆角: 9999px (rounded-full)
